# Class 


## Carculate 
- class 기본 형 
  - 클래스 이름과 기능들을 정하고 
  - 그에 맞게 작성 
  - __init__함수에 대해 배운 클래스 생성시 인수를 바로 받을 수 있음 
  - __name__함수는 py파일을 import 하느냐 직접 실행하느냐에 따라 return이 달라짐 
    - py 파일을 직접 실행 -> __main__출력 
    - import 시 -> 해당 py파일의 이름(실행하는 py 파일의 이름이 아닌)
